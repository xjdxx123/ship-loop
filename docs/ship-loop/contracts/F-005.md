## Edge cases owned
- GFM pipe-in-table-cell: a raw `|` in the Notes cell would open a 4th cell that GFM
  drops, silently truncating the rendered note; the row uses the `\|` escape — the only
  place the enum is constrained by table syntax. Conductor prose uses plain backticked
  values and never needs the escape.
- Charter without the row (every pre-F-005 charter, and profile charter-defaults that
  predate the key): step 3 pins row-absent = `merge`; old charters keep the exact
  pre-F-005 loop behavior with zero migration.
- pr mode with no `gh` binary, unauthenticated `gh`, or no `origin` remote → the
  feature parks (withheld-secret) instead of silently merging to main against the
  frozen charter.
- pr-mode rollback: nothing reached main, so `git revert` is meaningless — close the PR
  and delete the branch; pinned in step 6 so F-006's pr-mode branch cannot drift.
- Worktree/branch lifecycle in pr mode: the branch survives on origin after push, so
  worktree removal is safe; merge-mode lifecycle is byte-identical to today.
- Verdict quality in the PR body: step 5 already discards empty-`commandsRun` verdicts,
  so a created PR can never legally carry an evidence-free body.
